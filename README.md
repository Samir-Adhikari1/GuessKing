# 🎮 GuessKing

> **Memory + Logic = Mastery**

GuessKing is a memory-based word guessing game built with Django.  
Players must memorize words, then use logic to find the correct answer.

---

## 🧠 Core Idea

| Phase | Description |
|------|------------|
| 👁️ Memorize | Words shown briefly |
| 🧠 Recall | Words disappear |
| 🎯 Guess | Enter 5-letter word |
| 🔍 Analyze | Use feedback |

---

## 🎯 Objectives

| Goal | Description |
|-----|------------|
| 🧠 Memory | Improve recall ability |
| ⚙️ Logic | Apply reasoning |
| 🎮 Gameplay | Interactive experience |
| 📈 System | Scalable difficulty |

---

## 🛠️ Architecture

User → Django Views → Session → Templates → UI

---

## 🔁 Game Flow

Start → Difficulty → Memorize → Guess → Result

---

## 🧠 Core Logic

- Step 1: Correct letters  
- Step 2: Present letters  
- Step 3: Remaining absent  

---

## 🧩 Difficulty

| Level | Attempts | Multiplier |
|------|--------|-----------|
| Easy | 8 | ×1 |
| Medium | 7 | ×2 |
| Hard | 6 | ×3 |
| Expert | 5 | ×4 |
| Master | 5 | ×5 |
| Elite | 4 | ×7 |
| Legendary | 3 | ×10 |

---

## ⚡ Scoring

Score = Attempts Left × 10 × Multiplier
