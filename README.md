# 📖 Project Overview

> *“Memory + Logic = Mastery”*

**GuessKing** is an advanced **memory-based word guessing game** built with Django.

Unlike traditional Wordle:
- You **first memorize a word list**
- Then it disappears
- You must **recall + deduce the correct word**

This introduces a **cognitive layer (memory + logic)**, making gameplay deeper and more strategic.

---

# 🧠 Core Concept

The game combines two systems:

### 1. Memory System
- Words are shown briefly
- Player must remember them

### 2. Logic System
- Wordle-style feedback:
  - ✅ Correct position
  - 🟡 Present but misplaced
  - ❌ Not present
